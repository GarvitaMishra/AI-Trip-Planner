# ✈️ AI Trip Planner

An AI-powered travel planning web application built using the MERN stack.  
Users can generate smart travel itineraries, explore hotels, food places, transport options, estimated budgets, and manage trips with full CRUD functionality.

## 🚀 Features

- 🤖 AI-generated travel itineraries
- 🏨 Hotel recommendations
- 🍽️ Food & restaurant suggestions
- 🚖 Transport recommendations
- 💰 Budget estimation
- 📌 Save trips
- 📂 View all saved trips
- ❌ Delete trips
- 🌙 Dark mode support
- 🔐 JWT Authentication
- 📱 Responsive UI

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Axios
- Context API
- CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
